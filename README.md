
## 💻 Requirements
- Create an **virtual environment** for your code. If you're using VS CODE try:
```bash
  python -m venv venv
```
- Make sure your **virtual environment** is selected, then install the following libraries:
```bash
  pip install mysql-connector-python
```
```bash
  pip install tkinter
```
